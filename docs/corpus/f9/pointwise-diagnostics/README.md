# F9 Pointwise Diagnostics

本目录保存 pointwise reliability diagnostics 的执行总结，状态为 **历史主线 / 当前阻塞依据**。

| 文件 | 用途 |
|---|---|
| `execution-summary.md` | F3/F4/F9 pointwise 线每轮执行细节、测试命令、validation 指标和后续决策记录。 |

阅读顺序：

1. 先看 `../README.md` 的当前判定和 R0-R10 轮次表。
2. 需要追溯每轮执行细节时，再看 `execution-summary.md`。
3. 需要追溯计划原文时，看 `../plans/`。
